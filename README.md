## Getting Started

### Prerequisites
- Node.js 18+
- npm/yarn/pnpm/bun

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/MiniAiLive/facesdk-idsdk-online-demo-nextjs.git
2. Install dependencies:
   ```bash
   npm install
3. Run the development server:
   ```bash
   npm run dev
4. Open http://localhost:3000 in your browser
